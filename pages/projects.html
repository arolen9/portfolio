Showcase Projects

Home Lab Security Operations Center (SOC)

Objective: To build a functional, miniature Security Operations Center within a home lab environment to simulate real-world threat detection and incident response scenarios.

Methodology & Tools:

  • Deployed multiple VMs (Kali, Metasploitable) using VMware.
  • Configured a segmented network with a pfSense firewall for traffic control and logging.
  • Aggregated logs to a centralized Splunk instance for monitoring.
  • Simulated attacks (port scans, web exploits) using Metasploit from a Kali attacker VM.
  • Developed correlation searches and dashboards in Splunk to detect malicious activity based on the MITRE ATT&CK framework.

Outcome & Key Learnings:

Successfully detected and analyzed simulated attacks in real-time. Gained hands-on experience in SIEM administration, rule creation, and dashboarding. Demonstrated proficiency in log analysis, network traffic inspection with Wireshark, and understanding the attacker lifecycle.

Web Application Vulnerability Assessment

Objective: To perform a comprehensive vulnerability assessment on a target web application (OWASP Juice Shop) to identify and report security flaws based on the OWASP Top 10.

Methodology & Tools:

  • Used Nmap for port discovery and Burp Suite to map the application's structure.
  • Ran automated scans with OWASP ZAP to identify initial vulnerabilities.
  • Conducted manual testing for SQL Injection (SQLmap), XSS, and broken authentication.
  • Intercepted traffic with Burp Suite to check for sensitive data exposure.
  • Compiled a professional report detailing each vulnerability, its CVSS risk rating, and remediation steps.

Outcome & Key Learnings:

Identified multiple critical and high-severity vulnerabilities. Mastered the use of industry-standard tools like Burp Suite. Developed the ability to not only find vulnerabilities but also to articulate business impact and provide actionable mitigation steps.
